1. Lũy Thừa & Mũ

$\int x^\alpha dx = \frac{x^{\alpha+1}}{\alpha+1} + C \; (\alpha \ne -1)$

$\int \frac{1}{x} dx = \ln|x| + C$

$\int e^x dx = e^x + C$

$\int a^x dx = \frac{a^x}{\ln a} + C$

2. Lượng Giác Cơ Bản

$\int \cos x dx = \sin x + C$

$\int \sin x dx = -\cos x + C$

$\int \frac{1}{\cos^2 x} dx = \tan x + C$

$\int \frac{1}{\sin^2 x} dx = -\cot x + C$

Công Thức Hạ Bậc

$\cos^2 x = \frac{1 + \cos 2x}{2}$

$\sin^2 x = \frac{1 - \cos 2x}{2}$

$\tan^2 x = \frac{1}{\cos^2 x} - 1$

$\cot^2 x = \frac{1}{\sin^2 x} - 1$

Biến Đổi Tích Thành Tổng

$\cos a \cos b = \frac{1}{2}[\cos(a-b) + \cos(a+b)]$

$\sin a \sin b = \frac{1}{2}[\cos(a-b) - \cos(a+b)]$

$\sin a \cos b = \frac{1}{2}[\sin(a-b) + \sin(a+b)]$

3. Hàm Hợp $(ax+b)$

$\int e^{ax+b} dx = \frac{1}{a}e^{ax+b}$

$\int \cos(ax+b) dx = \frac{1}{a}\sin(ax+b)$

$\int \frac{dx}{ax+b} = \frac{1}{a}\ln|ax+b|$

4. Phân Thức & Lượng Giác Ngược

$\int \frac{dx}{x^2-a^2} = \frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right|$

$\int \frac{dx}{x^2+a^2} = \frac{1}{a}\arctan\frac{x}{a}$

$\int \frac{dx}{\sqrt{a^2-x^2}} = \arcsin\frac{x}{a}$

CÔNG THỨC TÍCH PHÂN TỪNG PHẦN

$\int u \, dv = uv - \int v \, du$

Thứ tự ưu tiên đặt u (Nhất Lô, Nhì Đa...)

  • Lô: Logarit ($\ln x, \log x$)
  • Đa: Đa thức ($x, x^2, 2x+1$)
  • Lượng: Lượng giác ($\sin x, \cos x$)
  • Mũ: Hàm mũ ($e^x$)
