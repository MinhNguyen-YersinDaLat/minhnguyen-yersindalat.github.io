### 2. Cách Thức Bổ Sung File (Quy Trình Hoạt Động)

Để hệ thống hoạt động trơn tru, bạn làm theo các bước sau:

**Bước 1: Chuẩn bị Thư mục**
* Tạo một thư mục chính (Ví dụ: `BaoTangToanHoc`).
* Đặt file `lichsutoan.html` (file chính) vào đó.
* Tất cả các file con ở danh sách trên **BẮT BUỘC** phải nằm cùng thư mục với file chính.

**Bước 2: Tạo Nội Dung File Con (Dựa trên mẫu)**
Thay vì viết code từ đầu, bạn hãy dùng `pitago.html` hoặc `so0.html` mà tôi đã cung cấp trước đó làm "khuôn mẫu".
1.  **Copy** file `pitago.html`.
2.  **Đổi tên** thành file bạn muốn làm (ví dụ: `newton.html`).
3.  **Mở file** `newton.html` bằng trình soạn thảo văn bản (Notepad, VS Code...).
4.  **Sửa nội dung văn bản:** Tìm các đoạn văn tiếng Việt trong đó và thay thế bằng câu chuyện của Newton.
5.  **Thay ảnh:** Tìm dòng chứa link ảnh (`src="..."`) và thay bằng link ảnh của Newton.

**Bước 3: Kiểm tra**
* Mở file `lichsutoan.html`.
* Tìm đến mốc Newton trên dòng thời gian.
* Bấm nút **"Chi tiết"**.
* Nếu cửa sổ hiện ra nội dung về Newton -> Thành công!

### 3. Cách Thêm Một Mốc Mới (Ngoài danh sách trên)

Nếu sau này bạn muốn thêm một nhà toán học mới không có trong danh sách (ví dụ: Hypatia), bạn làm như sau:

1.  **Tạo file HTML:** Tạo file `hypatia.html` (làm như Bước 2).
2.  **Khai báo trong file chính:**
    * Mở file `lichsutoan.html`.
    * Kéo xuống phần `<script>`, tìm biến `timelineData`.
    * Copy một đoạn mã `{...}` của một người khác.
    * Dán xuống vị trí bạn muốn (theo thứ tự năm).
    * Sửa lại các thông tin:
        * `year`: Năm.
        * `title`: Danh hiệu.
        * `figure`: Tên nhân vật.
        * `desc`: Mô tả ngắn.
        * `icon`: Tên icon (tìm trên FontAwesome).
        * `file`: Tên file HTML vừa tạo (`hypatia.html`).

Như vậy, bạn hoàn toàn có thể quản lý dự án này giống như quản lý các file văn bản mà không cần lo lắng quá nhiều về kỹ thuật lập trình phức tạp.
